---

## Java コードがきれいに表示できます

Marp を利用することで、Java のコードがキレイにハイライトされます

```java
public class SampleSlide {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

---

## Shell もきれいに表示できます

```bash
$ echo "Hello, World!"
$ javac SampleSlide.java
$ java SampleSlide
Hello, World!
```
